# Project: Portfolio Website
- Stack: React, Vite, Material UI (MUI), Framer Motion.
- Aesthetic: "Coastal Sophistication" (#023047, #006D77, #8ECAE6, #FFDD87, #102B3F, #062726)
- UI Elements: Heavy use of Glassmorphism (semi-transparent backgrounds with backdrop-blur). 
- Navigation: Right-aligned vertical scrollspy menu (do not use generic top navbars).
- Rule: Do not use default MUI styles. Always override with custom dark mode colors and glassmorphic effects.
